Poker Planning es una técnica de estimación que comunmente se usa para definir qué tan “pesado” o complejo es cada trabajo (ISSUE )del proyecto. 
Se hace normalmente con cartas de valores tipo Fibonacci, en honor a la famosa serie matematica (1, 2, 3, 5, 8, 13…), y sirve para llegar a un acuerdo rápido sin discusiones eternas.
Facilita mucho asignar tareas y coordinar equipos de trabajo.

Cada ISSUE de este proyecti cuenta con un Label "estimado:" seguido de un numero, entre mas alto el numero mas complejo es el componente.
